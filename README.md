# Homebrew tap

## Description

This is a custom Homebrew formulae repository (Tap) that hosts command-line tools developed by **israoo**. Through this tap, you can easily install and keep CLI tools updated using Homebrew.

### Featured projects

- **[TerraX](https://github.com/israoo/terrax)** - The interactive TUI executor for Terragrunt stacks

## Installation

### 1. Add the tap

First, add this repository as a Homebrew tap:

```bash
brew tap israoo/tap
```

### 2. Install TerraX

Once the tap is added, you can install TerraX by running:

```bash
brew install terrax
```

## Usage

After installation, you can use TerraX directly from the terminal:

```bash
terrax
```

For more information on how to use TerraX, check out the [project documentation](https://github.com/israoo/terrax).

## Updating

To update any tool installed from this tap:

```bash
brew upgrade terrax
```

Or to update all formulae:

```bash
brew upgrade
```
